# visitlocal
Projet personnel de fin d'année de BTS SIO SLAM 
Application web de découverte et de résérvation d'activités touristiques développée en Python avec Flask et SQLite.

Objectifs :
- consulter et filtrer les activités
- réserver une activité
- gérer ses réservations
- enregistrer des favoris
- publier des avis
- administrer les activités

Technologies utilisées : 
- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap

Fonctionnalités :
- authentification utilisateur
- espace admin
- réservations & historique
- favoris
- avis
- profil utilisateur
- upload d'images

Compétences développées :
- développement web
- base de données
- gestion de formulaires
- securité des mots de passe
- gestion de sessions
- débogage
